# JARRAH Rental React Frontend

A responsive ReactJS frontend application for a car rental service. The project includes Home, About, Cars, Contact, and Contract pages with search/filter and booking functionality.

## Features

- ReactJS single-page application using CDN React and React Router
- Home, About, Cars, Contact, and Contract pages
- Responsive layout for desktop and mobile
- Search and filter cars by brand or model
- Booking flow with car selection saved in localStorage

## Setup Instructions

1. Open the workspace folder in a web browser.
2. Open `frontend/index.html` directly in the browser.
3. Use the navigation menu to move between pages.

> If you want to publish to GitHub Pages, deploy the `frontend` folder as a static site.

## Files

- `frontend/index.html` — entry point and React CDN setup
- `frontend/app.jsx` — React application logic and pages
- `frontend/style.css` — responsive app styling
